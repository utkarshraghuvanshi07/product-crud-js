# 📦 Product CRUD JS

A simple **Product CRUD (Create, Read, Update, Delete)** application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

This project was created to practice core JavaScript concepts such as DOM manipulation, form handling, CRUD operations, and browser storage by building a dynamic product management interface.

> **Status:** 🚧 Under Development

---

## ✨ Features

* ➕ Create new products
* 👀 Display products dynamically
* ✏️ Update existing products
* 🗑️ Delete products
* 💾 Store products using **Local Storage**
* 🔄 Automatically load saved products on page refresh
* 📝 Basic form validation
* 🖼️ Display product images using image URLs
* ⚡ Dynamic UI rendering after every operation
